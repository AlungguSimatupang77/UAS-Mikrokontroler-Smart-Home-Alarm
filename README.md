# Sistem Alarm Keamanan Rumah Berbasis ESP32 Menggunakan Sensor PIR dan Blynk IoT

## Deskripsi

Proyek ini merupakan Proyek Akhir Mata Kuliah Mikrokontroler.

Sistem menggunakan ESP32 sebagai mikrokontroler utama, Sensor PIR sebagai pendeteksi gerakan, LED dan buzzer sebagai alarm, serta Blynk IoT sebagai media monitoring dan pengiriman notifikasi.

---

## Fitur

- Alarm ON/OFF melalui Blynk
- Sensor PIR Motion Detection
- LED Indicator
- Active Buzzer
- Motion Counter
- Status Alarm
- Push Notification
- Email Notification
- ESP32 + WiFi
- Wokwi Simulation

---

## Hardware

- ESP32 DevKit V1
- PIR Motion Sensor
- LED
- Resistor 220Ω
- Active Buzzer

---

## Software

- Arduino IDE
- Wokwi Simulator
- Blynk IoT

---

## Wiring

| Komponen | GPIO |
|----------|------|
| PIR | GPIO13 |
| LED | GPIO14 |
| Buzzer | GPIO12 |

---

## Cara Menjalankan

1. Buka project di Wokwi atau Arduino IDE.
2. Masukkan Blynk Template ID dan Auth Token.
3. Jalankan program.
4. Aktifkan Alarm melalui aplikasi Blynk.
5. Simulasikan gerakan pada sensor PIR.

---

## Hasil

Sistem berhasil:
